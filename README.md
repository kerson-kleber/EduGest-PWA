## 📱 Instalar no Celular (Android)

O EduGest funciona como um aplicativo instalável no seu celular, sem precisar da Play Store.

### ✅ Navegadores recomendados

| Navegador | Suporte PWA | Observação |
|---|---|---|
| **Microsoft Edge** | ✅ Excelente | Recomendado — opção "Adicionar ao telefone" |
| **Mozilla Firefox** | ✅ Bom | Menu → "Instalar" |
| Google Chrome | ⚠️ Pode apresentar tela branca | Não recomendado para este app |

> ⚠️ **Atenção:** O Google Chrome pode apresentar problemas de exibição (tela branca) com este aplicativo. Recomendamos o uso do **Microsoft Edge** ou **Mozilla Firefox**.

### Passo a passo — Microsoft Edge

1. Acesse o link do app no Edge
2. Toque nos **três pontinhos** (menu)
3. Toque em **"Adicionar ao telefone"**
4. Confirme a instalação
5. O ícone do EduGest aparecerá na sua tela inicial ✅

### Passo a passo — Mozilla Firefox

1. Acesse o link do app no Firefox
2. Toque nos **três pontinhos** (menu)
3. Toque em **"Instalar"**
4. Confirme a instalação ✅

## 🔄 Backup e Sincronização

O EduGest armazena os dados localmente no dispositivo. Para sincronizar entre celular e computador:

1. No dispositivo de origem → **Exportar backup** (gera um arquivo `.json`)
2. Envie o arquivo por e-mail ou WhatsApp
3. No dispositivo de destino → **Importar backup**

✅ Backup feito no celular abre no PC  
✅ Backup feito no PC abre no celular
